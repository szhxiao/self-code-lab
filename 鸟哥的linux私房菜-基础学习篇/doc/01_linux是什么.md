## 1.3 linux 的特色

### 1.3.1 linux 的特色

-   自由与开放的使用与学习环境
-   配备需求低廉
-   内核功能强大而稳定
-   独立作业

### 1.3.2 Linux 的优缺点

1. 优点

    - 稳定的系统
    - 免费或少许费用
    - 安全性、漏洞的快速修补
    - 多任务、多用户
    - 用户与用户组的规划
    - 相对比较不耗资源的系统
    - 适合需要小内核程序的嵌入式系统
    - 整合度佳且多样的图形用户界面

2. 改进

    - 没有特定的支持厂商
    - 游戏的支持度不足
    - 专业软件的支持度不足

3. 关于授权

    - Open Source
    - Close Source
